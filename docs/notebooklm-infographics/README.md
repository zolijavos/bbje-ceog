## Használati Útmutató

### 1. NotebookLM Beállítás

1. Nyisd meg a [NotebookLM](https://notebooklm.google.com/) oldalt
2. Hozz létre új notebook-ot "CEO Gala Infografikák" névvel
3. Töltsd fel a dokumentumokat:
   - `01-VENDEG-FOLYAMATOK.md`
   - `02-SZERVEZOI-FOLYAMATOK.md`
   - `04-ARCULATI-UTMUTATO-INFOGRAFIKA.md`

### 2. Custom Instructions Beállítás

A `03-NOTEBOOKLM-CUSTOM-INSTRUCTIONS.md` fájlból másold be a "Notebook guide" szekciót:

```
CEO Gala infografika stílus:
- Színek: monokróm alap + teal (#00A0A0) akcentus, arany (#C4A24D) csak VIP-nek
- Formák: szögletes, nincs lekerekítés
- Hangulat: elegáns, prémium, professzionális
- Kerülendő: élénk színek, játékos elemek
```

### 3. Infografika Generálás

1. Jelölj ki egy szekciót a dokumentumból (pl. "VIP Vendég Útja")
2. Kattints az "Infographic" gombra
3. A NotebookLM + Napkin AI automatikusan generál
4. Ha nem megfelelő, próbáld újra vagy finomíts a chatben

```
Fehér háttér:     #FFFFFF
Szürke háttér:    #F5F5F5
Szövegtörzs:      #4B5563
Címsorok:         #1F2937
Teal akcentus:    #00A0A0
Arany VIP:        #C4A24D
Siker zöld:       #059669
Hiba piros:       #DC2626
```

### Fő szabályok
- 90% monokróm + 10% akcentus
- Szögletes sarkok mindenhol
- Elegáns serif címsorok
- Minimalizmus, sok whitespace
- Arany CSAK VIP elemekhez

## Technikai Megjegyzések

### NotebookLM + Napkin AI infografika
- A NotebookLM a Napkin AI-t használja infografika generáláshoz
- A feltöltött dokumentumok tartalma alapján generál automatikusan
- Custom instructions befolyásolja a stílust
- Minden generálás egyedi - ha nem tetszik, próbáld újra

### Tippek a jobb eredményhez
- Strukturált, táblázatos dokumentumok jobban működnek
- Rövid, tömör szekciók könnyebben értelmezhetők
- Jelölj ki konkrét részt, ne az egész dokumentumot
- Ha túl színes: kérd chatben "használj monokróm színeket"
