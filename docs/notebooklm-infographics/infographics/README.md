## Használat

### 1. Feltöltés NotebookLM-be

1. Nyisd meg a NotebookLM-et
2. Hozz létre új notebook-ot
3. Töltsd fel a kívánt dokumentumo(ka)t

### 2. Infografika Generálás

1. Kattints a feltöltött dokumentumra
2. Válaszd az "Infographic" opciót
3. A Napkin AI generálja az infografikát
4. Ha nem tetszik, próbáld újra
